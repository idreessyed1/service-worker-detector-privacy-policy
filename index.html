Privacy Policy

Effective Date: March 2025

Author: Syed Idrees

The Service Worker Detector Chrome extension does not collect, store, or transmit any personal data or browsing activity.

  • ✅ No analytics
  • ✅ No tracking
  • ✅ No data is sent to any server

All functionality runs locally within your browser. The extension checks for a Service Worker on the current active tab only and displays the result in a popup UI. It does not persist or share any information.

If you have any concerns, feel free to reach out via the Chrome Web Store support page.
